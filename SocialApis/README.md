# SocialApis(α)
[Liberfy](https://github.com/atst1996/Liberfy)で複数のWebAPIへ接続することを目的に開発している.NET Framework向けのライブラリです。

### 対応サービス
以下の一部のAPIに対応しています。
- [Twitter](https://developer.twitter.com/)
- [Mastodon](https://github.com/tootsuite)
